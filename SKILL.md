# Praxis

Stigmergic intent-to-action substrate. Bridges knowing and doing.

Captures three continuous signal streams (conversation, artifacts, outcomes)
and feeds them into an NG-Lite learning substrate. The substrate's learned
topology handles classification, connection, surfacing, and learning
automatically.

Praxis does not generate code. It provides the context intelligence that
makes vibe coding reliable, repeatable, and self-improving.
